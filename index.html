The Biggest Boot

Firmware Updater For The Biggest Foot

  1. Plug The Biggest Foot in to Eurorack power via ribbon cable
  2. Plug a USB-C cable in to The Biggest Boot sub-board, and a desktop computer running Chrome.
  3. Click the Connect button.
  4. Select the "USB to UART Bridge Controller". If you don't see this option listed, try a different USB-C cable.
  5. Choose to erase & flash.
